对我来说，选品绝对是Top 1难题。
平时用卖家精灵、Sorftime看数据，这些工具本身没什么问题，数据也够全。但每次做选品调研，导出数据真的太麻烦了——一个一个下载，下完还要在Excel里折腾半天。

01

什么是Claude Code？为什么要用它？

先简单介绍一下。
Claude Code 是Anthropic官方推出的一个命令行工具，简单理解，它就是一个能帮你写代码、跑命令、做复杂任务的AI助手。
但你可能会问：豆包、DeepSeek这些通用AI不也能做分析吗？为什么要用Claude？
关键在于一个叫"Skill"的东西。
Claude Code有一个很厉害的特性——可以自定义"技能包"。你可以把一套完整的工作流程（比如"如何做亚马逊选品分析"）打包成一个Skill，里面包含数据获取、分析逻辑、报告格式等所有细节。
这样用的时候，你只需要说一句"分析这个品类"，它就会按照预设的专业流程走完所有步骤。
通用AI做不到这一点。你问豆包"帮我分析亚马逊市场"，它能给你一些建议，但它：
没有连接真实的市场数据API
不知道要分析哪些维度
生成不了专业的Excel和可视化报告
每次问的结果还不一样
而Claude的Skill就像一个"选品专家的完整工作流"，封装了一次，反复使用。
我花了一段时间对比了两者的区别，最后确定： 要做成这件事，Claude Code + Skill 是现在看下来更可行的方案。

03

它具体能做什么？

它具体的运作流程是这样的：
1. 市场大盘一目了然
打开Excel，第一个sheet就是整个品类的市场概况：
过去25个月的销量趋势
品牌数量变化
平均价格走势
竞争集中度指标（CR3、HHI这些专业指标都有）
2. 竞争分析全面到位
以前分析竞品，如果人工来做这件事的话，就看看BSR排行榜前面几个产品。现在：
Top 100产品的详细数据，每个产品的ASIN、价格、评论数、评分全部列出来
品牌集中度分析，哪些品牌在主导市场一目了然
3. 供应链对标
它会根据你选的品类，去1688上找相似产品，给你一个采购成本对照表。Amazon售价多少，1688采购价多少，预估毛利率多少，全部算清楚。
这样你就能快速判断这个品类有没有利润空间。
4. 一个评分模型告诉你能不能做
这是我最喜欢的部分。
它会用五维评分模型给整个品类打分：
市场规模（20分）
增长潜力（25分）
竞争烈度（20分）
进入壁垒（20分）
利润空间（15分）
总分100分，一眼就能看出这个品类值不值得进。
